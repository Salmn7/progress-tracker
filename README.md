# Momentum — Progress Tracker

A modern, minimal, and fully client-side productivity dashboard for tracking progress across fitness, learning, health, work, finance, and creative goals.

Built using vanilla HTML, CSS, and JavaScript.

---

# Features

## Dashboard
- Real-time overview of:
  - Total entries
  - Weekly activity
  - Goal completion
  - Streak tracking
- Recent activity feed
- Goal progress summaries

## Activity Logging
- Add custom progress entries
- Categorize activities
- Add notes and mood tracking
- Edit and delete entries
- Search and filter logs
- Bulk selection and deletion

## Goal Management
- Create goals with:
  - Target values
  - Deadlines
  - Progress tracking
- Update progress dynamically
- Goal completion indicators
- Deadline monitoring

## Analytics
- Weekly activity visualization
- Category breakdown analysis
- Productivity insights
- Heatmap activity tracking
- Progress statistics
